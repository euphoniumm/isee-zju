【科目】：通信原理
【作者】：Emma

【课程学习】：我上的是刘安老师的英文班。作为信电的四大天书之一，学起来确实有一定的难度。再加上英文班，老师全英文讲课，对于我这个英语渣渣，就需要自己课下再用功一点了，不过英文班讲的内容还是相对来说比较浅的，只要认真学基础的东西还是可以搞懂的。书中对应的老师讲过的内容要仔细看看，对于理解以及考试都比较有用。老师也很好，会讲的比较细，也会有两次大课专门给同学答疑，会事先通知让每个同学准备三个问题。无论什么问题刘老师都会很认真的解答！
【作业】：一共就四次作业，不过一次的作业量有一点大，老师也会提前布置，尽量不要拖到要交的时候才做，因为作业里面有一些内容老师课上没有讲过，需要自己看书来做。对于作业的给分，助教还是比较严格的，尽量不要空着题目吧，然后也不要抄答案！不要完全和答案一样！还有就是答案也有有错的地方。作业一定一定要好好做，考试题型和作业类似的！
【小测】：正常应该是有三四次小测，但是我们只小测了一次，剩下的都是用每个人提三个问题的方式代替了。
【期中考试】：主要考模拟通信部分，ch1-ch6.题目和作业题很相似，作业题中涉及到老师没讲过的内容的考试不会考。题型是不定项选择、判断、计算题。选择还是比较简单的，就是基本的公式代一代，基本的概念理解好就可以。判断题我记得还是会有那么两个有点不确定的，主要是对于一个概念的理解吧。计算题比较简单，和作业题类似。
【期末考试】：只靠数字通信的部分，ch7以及之后的内容。题型是不定项选择、判断、计算。期末的判断题会要求你如果错了需要解释原因，所以一道选择题的分数比较大，但是我个人觉得比期中的小选择题更简单，基本不会不确定自己做的对不对。期末考试的计算题会比期中难一些，但是也还好，主要是有一些坑，一定要仔细读题，理解好问题的条件。我个人觉得数字调制还是要比模拟调制部分难一些的，公式比较多，类型比较多，所以不要模型搞混。
【给分】很好
